# UTP Library

Sistema de gestión bibliotecaria desarrollado como proyecto académico para la Universidad Tecnológica del Perú.

## Tecnologías utilizadas

- Java 21
- JavaFX
- MySQL
- Maven
- Git
- GitHub

## Funcionalidades

- Inicio de sesión
- Gestión de usuarios
- Gestión de libros
- Gestión de préstamos
- Gestión de reservas
- Administración de horarios

## Arquitectura

El proyecto fue desarrollado siguiendo una arquitectura modular e implementando los patrones de diseño:

- Singleton
- Factory
- Repository
- Facade
- Observer
- State
- Decorator

## Autor

Proyecto desarrollado como parte del curso de Ingeniería de Software.
